# AI Usage Declaration

| Tool | Purpose | Supported project areas |
|---|---|---|
| ChatGPT | Research explanation, pipeline planning, code drafting, debugging, result interpretation, report and slide structuring | Proposal, methodology, scripts, notebooks, report language, presentation |
| GitHub Copilot / coding assistants, if used locally | Code completion and refactoring | Python scripts and notebooks |

## Control and verification

AI-generated suggestions were not treated as empirical evidence. Numerical
claims were taken from executed analysis outputs. Important decisions were
checked through:

- data-quality audits;
- saved exclusion logs;
- internal clustering metrics;
- random-seed stability;
- bootstrap stability;
- leave-one-city-out validation;
- leave-one-recording-out validation;
- post-cluster analysis in original physical units.

The group remains responsible for the final code, interpretation, citations,
and presentation.
